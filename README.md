## 网易严选

### 项目描述

1. 此项目为一个关于电商的WebApp(SPA)
2. 包括六个主路由:首页,识物,分类,购物车,个人,登录/注册
3. 将要使用Vue技术栈、ES6、webpack等前端最新技术
4. 采用模块化、组件化、工程化的模式开发


### 项目的编码阶段

#### day01路由的搭建及静态页面

1.搭建好了环境和路由

2.开始写静态页面

- 导航的宽度不能自动撑开

  加最小宽度 min-width：100%

- better-scroll不滚动的问题

  内层容器与外层容器一样高，导致不能滑动，解决办法，为外层容器设置宽高都是100%
